# GlaMaterials

Односторінковий інструмент: накладає фірмовий водяний знак GlamEng на PDF або фото за один клік.

Домен/бренд: **GlaMaterials** (GlamEng).

## Як це працює

1. Користувач перетягує або обирає файл(и) — PDF (будь-яка кількість сторінок) або фото (JPG/PNG/WEBP). Кілька файлів об'єднуються в один PDF по порядку.
2. Кожна сторінка/фото вписується в аркуш A4 (масштаб "contain" — без спотворень; якщо пропорції не A4, з боків/зверху-знизу додаються білі поля, щоб шаблон водяного знаку ліг рівно).
3. Поверх накладається `assets/watermark.jpg` (A4, 1414×2000) з прозорістю, яку можна підкрутити повзунком (за замовчуванням 15%) — прев'ю оновлюється миттєво.
4. Одна кнопка «Завантажити PDF» збирає фінальний файл і одразу віддає його браузеру на завантаження.

**Все відбувається локально в браузері.** Файли ніколи нікуди не надсилаються — немає бекенду, немає сховища. Життєвий цикл: завантажив → обробилось → скачав. Нічого не залишається "всередині" застосунку між сесіями.

## Структура

- `index.html`, `styles.css`, `app.js` — весь застосунок.
- `assets/watermark.jpg` + `assets/watermark-data.js` — водяний знак, вшитий як base64 (щоб не було зовнішніх запитів).
- `vendor/` — `pdf-lib` (збірка PDF) та `pdf.js` (рендер вхідних PDF-сторінок), завантажені локально, без CDN у рантаймі.
- `vercel.json` — кеш-заголовки для статики.

Ніяких залежностей для збірки — це чистий статичний сайт, працює просто відкриттям `index.html` або з будь-якого статичного хостингу.

## Локальний запуск

```bash
python3 -m http.server 8080
# відкрити http://localhost:8080
```

## Деплой на Vercel

Найпростіше — так само, як інші сайти в цьому акаунті: підключити GitHub-репозиторій до Vercel, і кожен push у `main` буде деплоїтись автоматично.

1. Створити репозиторій на GitHub (наприклад `glamaterials`) і запушити цей код.
2. На [vercel.com](https://vercel.com) → **Add New Project** → обрати цей репозиторій.
3. Framework Preset: **Other** (це статичний сайт, білд не потрібен — Build Command і Output Directory можна лишити порожніми).
4. Deploy.

Після цього Vercel видасть посилання на кшталт `glamaterials.vercel.app`, яким можна одразу ділитися. За бажанням — прив'язати кастомний домен через Vercel → Settings → Domains (аналогічно до `saenko-adv.com.ua`).

## Налаштування прозорості за замовчуванням

Змінна `DEFAULT_OPACITY` у `app.js` (зараз `15`, тобто 15%).
